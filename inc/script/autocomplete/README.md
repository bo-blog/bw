[jQuery.AutoComplete](http://work.jiani.info/jQuery.AutoComplete/)
=================
jQuery.AutoComplete是一个基于jQuery的自动补全插件。借助于jQuery优秀的跨浏览器特性，可以兼容Chrome/IE/Firefox/Opera/Safari等多种浏览器。

特性一览：

* 支持补全列表的宽度设定。
* 支持补全列表的最大高度设定。
* 支持补全列表的行数限制。
* 支持补全列表的显示位置及方向的设定。
* 支持自定义匹配规则。
* 支持匹配文本的渲染。
* 支持自定义匹配文本的渲染样式。
* 支持补全列表的样式设定。
* 支持自定义补全列表项的创建。
* 支持多种数据源。
* 支持'json'和'xml'两种数据格式。
* 支持异步处理。
* 支持错误调试。
